Projecten/Vaardigheden


Dit is mijn eerste project. Het doel was om een website te bouwen over jezelf met de juiste informatie. Zoals u ziet is het aardig gelukt en werkt alles goed. Om dit te kunnen maken heb je natuurlijk vaardigheden nodig die ik bezit. De vaardigheden die ik bezit en kan zijn de programmeertalen HTML CSS en Javascript. Ik ben het beste in HTML CSS. HTML CSS is Front End. Persoonlijk vind ik Front End ook leuker dan Javascript (Back End). Je hoort ook goed Engels te kunnen, omdat dat een hele belangrijke taal is die je moet beheersen. Dat komt omdat alle tags Engels zijn en je het dus goed begrijpen moet. Hieronder zie je een van mijn eerste opdracht: een mondriaan maken. Ik zelf vind de mondriaan goed gelukt. Uiteraard hebben we meer design opdrachten gekregen die ik allemaal succesvol afgemaakt heb.
